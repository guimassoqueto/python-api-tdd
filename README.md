# python-api-tdd

This is a Python TDD API built using Django.


##  How to init

1. Set the .env file:
```shell
make env
```

2. Make data migrations:
```shell
make up
```

3. Create a superuser:
```shell
make superuser
```

4. Login with your previously superuser credentials: [localhost:8000/user](http://localhost:8000/admin)
